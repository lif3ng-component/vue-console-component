## props

### title
- Type: `string`
- Default: `'基本信息'`

头部显示文字

### hideLabel
- Type: `boolean`
- Default: `false`

隐藏 label 区域

### noBorder
- Type: `boolean`
- Default: `false`

隐藏边框。隐藏后背景色自动置为透明，隐藏头部，`title` 不再显示。

### enterSubmit
- Type: `boolean`
- Default: `false`

回车键触发 `form 组件` 提交。

### items
- Type: `array`
- Default: `[]`

`form 组件` items

### rules
- Type: `object`
- Default: `{}`

`form 组件` rules

### v-model
- Type: `object`
- Default: `{}`

`form 组件` v-model 双向绑定对象
